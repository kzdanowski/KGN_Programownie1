9. Napisz rekurencyjną funkcję potega(a,n),
która obliczy a do potęgi n, korzystając
z rekurencyjnej zależności:
potega(a, 2*n) = potega(a,n)^2
potega(a, 2*n+1) = a*potega(a,n)^2
Porównaj, przy pomocy timeit, 
czas działania tej funkcji 
z czasem działania funkcji
def potega_iter(a,n):
    if n < 0:
       return -1
    wynik = 1
    for i in range(n):
        wynik *= a
    return wynik
