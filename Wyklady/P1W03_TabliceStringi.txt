1. Lista

Lista to struktura danych, w której elementy są ułożone 
liniowo.  W Pythonie mamy dostęp w stałym czasie do i-tego elementu listy.
Możemy zmieniać elementy listy - są one ,,mutable''

Listy tworzymy:
lista = [1, 2, "Ala", "ma", "kota"]

Listy indeksujemy od zera.
Element i listy pobieramy przez lista[i]
Elementy list można zmieniać, np.
lista[2] = "Ola"
lista[-1] - ostatni element listy

Podlisty
lista[i:j] = lista od i-tego do (j-1) elementu
Uwaga.Reprezentując lista[i:j] Python tworzy kopię listy. 

lista.append(element) - dodaje element na końcu listy
lista.extend(inna_lista), lista += inna_lista dodają na koniec lista
                                   listę inna_lista
                                   lub dowolny obiekt iterowalny.
lista1+lista2 łączy dwie listy (konkatenuje).
lista1+=lista2 - dołącza do lista1 lista2

Pytanie.
x = [1,2]
Jaka jest różnica pomiędzy:
x+=[3,4]
x.append([3,4])

lista.remove(elem) usuwa pierwsze wystąpienie elem na lista.
Uwaga. Jeśli elem nie występuje w lista mamy błąd.
lista.pop(index) usuwa element o indeksie index

"elem in lista" sprawdza, czy element jest na liście.

Często mamy dwie wersje operacji na listach.
x = [1,2,3, 9,3,5,4]
sorted(x) - sortuje kopię listy x
x.sort() - sortuje listę x

x.sort() to wywołanie metody listy x, działa ona na liście,
sorted(x) - wykonuje operację sortowania na kopii listy x

Tworzenie list.
Wyróżnianie.
list1 = [i for i in range(2,8)]
list2 = [i%3 for i in range(2,8)]
list3 = [(i,j) for i in range(2,8) for j in range(3,6)]
Co oznacza 
list4 = [[(i,j) for i in range(2,8)] for j in range(3,6)]

3. Zbiory
Tworzymy je przez {el1, el2, ..., el3}
albo używając wyróżniania 
zbior = {i for i in range(9)}

zbior.add("ala")
zbior.remove(3)

"ela" in zbior 
W zbiorach mogą znajdować się tylko elementy immutable: n-tki, liczby, ciągi znaków.
Nie mogą znajdować się: listy, zbiory.
